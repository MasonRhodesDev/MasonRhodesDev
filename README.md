## ⬇️ Run It Yourself

Most of the above ships as packages. Arch users can add [my pacman repo](https://github.com/MasonRhodesDev/arch-repo):

```ini
[mason]
SigLevel = Optional TrustAll
Server = https://masonrhodesdev.github.io/arch-repo/x86_64
```

then `sudo pacman -S hyprstate sni-watcher ...` as usual. Works on Steam Deck too, via [deck-tenant](https://github.com/MasonRhodesDev/deck-tenant)'s rootless pacman root.
